  • What Is It?

What is Mesquite Mocap?

Mesquite Mocap is a affordable motion capture device developed by a group of individuals at ASU that has real-time capture capabilities. Some of the benefits of Mesquite Mocap include:

Affordability

One of Mesquite Mocap's signatures is its extremely affordable price point due to the unique set of hardware we will be using for the motion capture system. This means that anyone, from a enthusiast to industry professional, can use it.

Real-Time Capture

Our Mocap system's propreitary hardware & software allow for accurate & reliable real time capture capabilities, so it can be used for almost any application with reliability.

Extendability

Because of the specific hardware & configurations we use in our Mocap systems, users have the option to potentially alter the hardware in the system in a variety of manners.

Open Source Software

All of our code is open source, which means anyone familiar with the tools & languages we use can alter the software for the mocap system to best fit their own use case(s).

Scalability

Mesquite Mocap's systems have the potential to be scaled affordably due to the specific hardware we use while we also have open source software. This allows for almost any interested user to have the opportunity to use our system.

Do It Yourself(DIY)

The DIY potential for our Mocap systems are virtually limitless, with users being able to customize parts & software functionally and cosmetically as they would like or best see fit.
